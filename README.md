# Global-Seismic-Safety-Monitor.
Finds data on earthquakes around the world and lets you see the information in real time

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download all files
2. Run index.HTML on live server

Instructions for using the software:

1. Scroll around map
2. Click a dot to see the info on that reported earthquake
3. Click "more info" to go to the offical USGS earthquake site about that earthquake

## Development Environment

To recreate the development environment, you need the following software and/or libraries:

* Must be able to run HTML, CSS, and JavaScript
* Must have a live server or a .NET local host
